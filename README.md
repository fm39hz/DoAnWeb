## Sử dụng

### Client side

#### Cài đặt

Di chuyển tới Thư mục chứa client

```bash
cd WebClient
```

Cài đặt dependencies

```bash
pnpm install
```

#### Khởi chạy client

Khởi chạy tại <http://localhost:5173>.

```bash
pnpm run dev
```

#### Build

```bash
pnpm run build
```

#### Chạy Test

Test bằng terminal

```bash
pnpm run test
```

Test bằng ui

```bash
pnpm run test:ui
```
